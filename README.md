### Clone project:
```sh
git clone git@github.com:kirillchistov/python-project-49.git
```

### Install:
```sh
cd python-project-49
```
```sh
make install build package-install
```

### Launch games:
```sh
 make brain-even
```
```sh
brain-calc 
```
```sh
brain-gcd 
```
```sh
brain-progression 
```
```sh
brain-prime
```
